\# 03 - User Stories and Acceptance Criteria



\## US-1: Student Registration



As a student,

I want to create an account,

So that I can access courses.



Acceptance Criteria:

\- Student must provide name, email, password

\- Email must be unique

\- Password must be minimum 8 characters

\- Account should be created successfully



---



\## US-2: Student Login



As a student,

I want to login securely,

So that I can access my dashboard.



Acceptance Criteria:

\- User must enter correct email and password

\- Error message shown for invalid credentials

\- Successful login redirects to dashboard



---



\## US-3: Search Courses



As a student,

I want to search courses by name or category,

So that I can find relevant courses.



Acceptance Criteria:

\- Search bar should be available

\- Results should match keyword

\- Empty result message should be displayed if none found



---



\## US-4: Enroll in Course



As a student,

I want to enroll in a course,

So that I can start learning.



Acceptance Criteria:

\- Student must be logged in

\- Course must have available seats

\- Enrollment confirmation should appear

\- Enrollment record must be stored in database



---



\## US-5: Instructor Creates Course



As an instructor,

I want to create a new course,

So that students can enroll.



Acceptance Criteria:

\- Instructor must be logged in

\- Course title and description are required

\- Course should be visible in catalog after creation
